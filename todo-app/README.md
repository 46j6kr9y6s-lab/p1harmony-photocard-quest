# To-Do List Application 📝

A modern, fully functional to-do list application with local storage persistence. Never lose your tasks!

## ✨ Features

### Core Functionality
- ✅ **Add Tasks** - Easily add new tasks with just a few clicks
- ✅ **Mark Complete** - Check off tasks as you complete them
- ✅ **Delete Tasks** - Remove individual tasks
- ✅ **Clear Completed** - Bulk delete all completed tasks
- ✅ **Clear All** - Remove all tasks at once

### Smart Filtering
- 📋 **All Tasks** - View everything
- 🔄 **Active** - Only show incomplete tasks
- ✅ **Completed** - Only show finished tasks

### Data Management
- 💾 **Local Storage** - Your tasks are saved automatically in your browser
- 🔄 **Persistent** - Tasks survive browser refresh
- 📊 **Statistics** - Track total and completed tasks
- 🎯 **Priority Levels** - Tasks have priority badges (High, Medium, Low)

### User Experience
- 🎨 **Beautiful UI** - Modern gradient design with smooth animations
- 📱 **Responsive** - Works perfectly on desktop, tablet, and mobile
- ⌨️ **Keyboard Support** - Press Enter to add tasks
- 🎯 **Instant Feedback** - Real-time updates and confirmations

## 🚀 How to Use

1. **Open the app** - Open `index.html` in your browser
2. **Add a Task** - Type in the input field and click "Add" or press Enter
3. **Mark Complete** - Click the checkbox next to a task
4. **Delete Task** - Click the Delete button
5. **Filter Tasks** - Use the filter buttons to view different task types
6. **Clear Tasks** - Use the clear buttons to remove multiple tasks at once

## 📁 Files

- `index.html` - HTML structure
- `style.css` - Beautiful styling and animations
- `script.js` - App logic and local storage management

## 💾 Local Storage

Your tasks are automatically saved to your browser's local storage. This means:
- Close the browser? Your tasks are still there!
- Refresh the page? Everything is preserved!
- No account needed, no cloud required!

## 🎨 Customization

You can easily customize:
- Colors in `style.css`
- Priority levels in `script.js`
- Task confirmation messages in `script.js`

## 🔧 Technical Details

- Pure JavaScript (no frameworks)
- LocalStorage API for persistence
- XSS protection with HTML escaping
- Mobile-first responsive design
- Smooth CSS animations
